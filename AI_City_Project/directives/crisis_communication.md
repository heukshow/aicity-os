# 📋 Directive: 위기 관리 및 이해관계자별 커뮤니케이션 SOP

## 🎯 목표
도시 프로젝트 내 위기 상황(지연, 클레임 등) 발생 시, 신뢰를 유지하고 해결 중심의 투명한 소통을 통해 '신뢰 지반'을 복구함.

## 🛠️ 위기 복구 공법 (Crisis Response Protocol)
1.  **현황 인정 (Status Recognition)**: 지진(에러) 발생 사실을 숨기지 않고 즉시 투명하게 공개.
2.  **원인 분석 (Root Cause)**: 구조적 결함인지, 외부 노이즈인지 명확한 데이터 기반 분석 제시.
3.  **해결 계획 (Architecture Plan)**: 임시 보강 공사(Workaround)와 근본 보수(Permanent Fix) 일정 제시.
4.  **예방책 (Future Proofing)**: 동일한 지진에 무너지지 않도록 내진 설계(프로세스) 강화 방안 포함.

## 📤 출력 (Outputs)
- `AI_City_Project/outputs/crisis_communication_scripts.md`: 이해관계자별(상급자/팀원/고객) 맞춤 메시지 초안
