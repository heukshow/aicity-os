# 📋 Directive: 비개발자 업무 자동화 및 생산성 혁신 SOP (No-Code Automation)

## 🎯 목표
코딩 지식이 없는 구성원도 AI와 노코드(No-code) 도구를 활용하여 일상적인 반복 업무를 자동화함으로써, 휴먼 에러를 방지하고 고부가가치 창업/기획 업무에 집중할 수 있는 '초효율 업무 환경'을 구축함.

## 🛠️ 자동화 시공 공법 (The Automation Build)
1.  **Blueprint (업무 분해 및 식별)**: 하루/주간 단위의 업무를 '판단이 필요한 일'과 '규칙적인 반복 일'로 구분하고, 자동화 대상(Logic-based) 식별.
2.  **Utilities (노코드 도구 선정)**: ChatGPT(AI), Zapier(연결), Google Sheets(데이터), Notion(정리) 등 비개발자 친화적인 도구 세트 구축.
3.  **Infrastructure (워크플로우 설계)**: 입력(Trigger) -> 처리(Action) -> 결과(Output)로 이어지는 자동화 파이프라인 설계.
4.  **Landscape (단계적 도입)**: 처음부터 거창한 시스템이 아닌, 10분 단축되는 작은 업무부터 시작하여 성공 레퍼런스를 쌓는 '모듈형 확장'.
5.  **District (유지보수 및 전파)**: 자동화 프로세스를 문서화하고, 팀 내부에 공유하여 조직 전체의 '디지털 지능(Digital IQ)' 상향 평준화.

## 📤 출력 (Outputs)
- `AI_City_Project/outputs/no_code_automation_ideas.md`: 비개발자용 업무 자동화 방안 10선 및 구현 가이드
